Integração com a Super Promoção

Disponibilizamos acesso à Super Promoção or meio dos dois endpoint listados abaixo.

  POST     Retorna a lista de coupon (produtos disponíveis para resgate) de um usuário específico.

/v2/supersale/check-coupon/

  • Parâmetros de Entrada

    Este endpoint fornece a lista de produtos que um cpf tem para resgatar em um estabelecimento participante da Super Promoção. Deve-se enviar o objeto da Tabela 22 no corpo da requisição.

    Objeto Parâmetro Descrição
    CouponCheck - -
    string cpf CPF do usuário para checagem

    Tabela 22

  • Resposta ( status_code = HTTP_200_OK )

    A Tabela 23 mostra a referência aos parâmetros inclusos na resposta desse endpoint.

    Objeto Parâmetro Descrição
    Product - -
    string
    		 cpf CPF do usuário
    string
    		 product_code Código do Produto (EAN ou SKU)
    string
    (decimal)     		max_amount Quantidade máxima

    Tabela 23

  POST     Marca um coupon como utilizado.

/v2/supersale/confirm-coupon/
  • Parâmetros de Entrada

    Por meio desse endpoint é possível marcar como usados os coupons de um determinado CPF. É necessário o envio do objeto descrito na Tabela 24 no corpo da requiisição.

    Objeto Parâmetro Descrição
    CouponConfirmUse - -
    string cpf CPF do usuário para checagem
    CouponProduct products -
    string product_code Código do produto (EAN ou SKU)
    string
    (decimal)     		amount Quantidade do produto

    Tabela 24

  • Resposta ( status_code = HTTP_200_OK )

    A Tabela 25 mostra a referência aos parâmetros inclusos na resposta desse endpoint.

    Objeto Parâmetro Descrição

    string cpf CPF do usuário para checagem

    string product_code Código do produto (EAN ou SKU)
    string
    (decimal)     		amount Qauntidade do produto
