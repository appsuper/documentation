API de Estabelecimentos

Na plataforma do App Super, cada parceiro pode ter diversos estabelecimentos. Nessa API você pode ter acesso às informações refentes aos mesmos.

  GET     Retorna lista de estabelecimentos do parceiro.

/market/

  • Parâmetros de Entrada

    Se necessário é possível fornecer como dado de entrada para esse endpoint um parâmetro querystring de paginação para o resultado, por padrão os resultados são mostrados com paginação de 100 itens.

  • Resposta ( status_code = HTTP_200_OK )

    A Tabela 1 mostra a referência aos parâmetros inclusos na resposta desse endpoint.

    Objeto Parâmetro Descrição
    Pagination - -
    integer count Número de Itens por página
    string
    (URI)     		next Url da página seguinte
    string
    (URI)     		previous Url da página anterior
    list [Market] results Lista de estabelecimentos
    Market - -
    string (uuid) id Identificador único do estabelecimento
    string company_name Nome empressarial
    string trading_name Nome Fantasia
    string
    (URI)     		logo Logomarca do estabelecimento
    string
    (URI)     		image Foto de fachada do estabelecimento
    string cnpj cnpj do estabelecimento
    string full_address Endereço completo
    string city cidade
    string
    (Enum)    		 state Estado
    string street Rua
    integer number Número do establecimento na rua
    string district Bairro
    string zipcode CEP
    string complement Complemento do endereço
    string coordinate Coordenadas no GPS
    string email Email do Estabelecimento
    string phone Telefone do Estabelecimento
    boolean conference_required Processo de conferência obrigatório [2]
    boolean password_required_cancelling_order Requer senha de gerente ao cancelar um pedido via ocorrência
